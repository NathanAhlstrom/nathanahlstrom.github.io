Using the web, even for mundane uses like email, is like a swim in shark infested waters for a variety of reasons and even because of the advertising on these sites.

The ads are distracting at best, pornographic at worst.

For the Christian we need to have our eyes fixed on the Son of God (Colossians 3:1-2) and concerned about our heart motives related to our internet usage.

Installing an Ad blocking browser extension for the Firefox web browser can be very helpful.  Here is how to get it working:

  1. Install Firefox from firefox.com
  2. Optionally set Firefox as your default browser.  [Note: it will bug you to do this, say No if you don't want it to be the default and tell it to not ask again.]
  3. Install AdBlock Edge from the Firefox Add-ons Page (See image below)
  4. Restart Firefox.
  5. You will see the AdBlock Edge first run page.  It is safe to close this tab on your browser.
  6. You are all set to be ad-free.

This solution can be a helpful tool in the toolbox, but more than another tool we all need our hearts to be changed from pursuing sin to loving Jesus.
